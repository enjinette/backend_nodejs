# Installation Guide

1. `npm install`
2. `node index.js`

**NOTE: backend_nodejs/index.js**

> Line 12: force is set to true to drop existing database.

# Environment Variables
| KEY | VALUE |
| ----------- | ----------- |
| HOST | localhost |
| DB_USER | root |
| DB_PASSWORD | admin |
| DB | usersdb |
| PORT | 8000 |
